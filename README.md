## 🧩 Working With CSS Mods

CSS mods change how Jellyfin looks. You can use them to adjust colors, spacing, layout, and other visual parts of the web app.

Good CSS mods can help you:

- Change the page style
- Hide items you do not use
- Make text easier to read
- Match the app to your setup

Keep CSS files named in a way that makes sense, such as `dark-theme.css` or `compact-layout.css`.

## 💻 Working With JavaScript Mods

JavaScript mods change how Jellyfin behaves in the browser. These can add small tools, change page actions, or improve parts of the web app.

Good JavaScript mods can help you:

- Change page behavior
- Add small helper tools
- Adjust how buttons work
- Improve your daily use of Jellyfin

Keep JavaScript files named clearly, such as `quick-actions.js` or `menu-fix.js`.

## 🔒 Safe File Practices

Before you change anything, it helps to keep a backup of your Jellyfin files. That way you can go back if you want to undo a change.

A simple backup plan:

- Copy the original files to a backup folder
- Save a date in the folder name
- Keep each mod set in its own place
- Check file names before replacing anything
